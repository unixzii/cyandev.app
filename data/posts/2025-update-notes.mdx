# 解锁新成就：驾驶

记得曾经说过自己是不会学开车的，是的，大学毕业 5 年了我还没有驾照。很多朋友都在劝我有时间考一个驾照吧，而我每次都说下次一定。因为比较宅，一直没有什么重度开车的需求，加上曾经在家自学倒车入库的挫败经历，导致自己一直没有一个驱动力去学车。但今年因为想换个地方租房，发现通勤距离成为了灵活选择的一把枷锁。通勤方便、品质好和便宜成为了租房的不可能三角，于是我决定舍弃通勤方便，考个驾照吧！

有的时候确实需要一些激情消费，而我在车的事情上激情消费了两次。因为我其实有很严重的拖延症，即便打算学车了，也一直没付诸实践。而有次朋友正好听说我打算学车，推荐我去一个驾校看一看，结果看着看着就报名了… 既然都报名了，那就只能开始学咯。学车的经历我之前也写了篇[文章](/post/take-driving-license)记录。

买车的时候也是激情消费，本来当时就是出去逛街吃饭，然后路过特斯拉门店的时候进去逛了一下，结果不知道怎么就在店员和朋友的怂恿下把定金交了。更离谱的是，我当时还没拿到驾照，提车的时候只能让对象开回来。

![New Car](https://images.cyandev.app/2025-update-notes/IMG_1696.jpg "New Car;3099x2324;L3CGY{bw00og009G_3%M4T~q.9Dj")

开车这件事对于我来说就一个词 —— 真香。本来还在算计成本什么的，但它带来的灵活性和自由我觉得是很难估量的。很多时候，出门与否真的就看有没有车。

至于怎么选车，因为当时没有驾照不能试驾，加上本来也比较喜欢特斯拉的设计风格，就直接盲订了 Model 3。当时恰好有促销活动，选的长续航 + 白色内饰，整体落地不到 30 万，加上 5 年免息和政府补贴真的非常划算。比较意外的是特斯拉的驾驶体验的确很优秀，让我慢慢也开始享受驾驶了。

虽然还是一个新手司机，但我在这一年的时间里，0 事故 0 扣分地安全行驶了一万多公里。可能是因为实习期怕扣分的缘故，我的驾驶风格也十分保守，几乎没有超速和压实线变道。就希望能赶紧安稳地把实习期给渡过了，接着就可以做五星好市民了（开玩笑）。

最后，如果你问我到底应不应该买车，我的回答很简单：如果经济压力不大，并且也**不讨厌**开车的话，那就无脑买吧，因为它解锁的真的是另一种生活方式。

# Home Sweet Home

今年告别住了 4 年的 Loft，也搬到良渚当“村民”了。新房是个平层，比之前大了很多，但因为地理位置比之前还便宜。四室一厅两卫，房东没有购置任何家具，给了我们充分的 DIY 空间。地库设计对充电桩很友好，特斯拉官方赠送的线材距离完全够用，有了家充之后出行就更无压力了。

![New Home](https://images.cyandev.app/2025-update-notes/IMG_3691.jpg "New Home;4032x3024;LGJG~10g4o%2.TrBD*tS9~E2NG?a")

对于日常生活来说，平层比 Loft 真的好太多。上下层虽然感觉空间大一些，但很多时候并不方便。因为从楼梯上搬东西麻烦，每个屋子的功能也相对固定。而现在的平层在总体面积差不多的情况下，空间反而显得宽敞了很多，也更有生活的氛围了。

顺便也想推荐一些生活好物，今年购入对生活品质提升最大的东西，我认为就是洗烘一体机了。之前洗衣服经常需要考虑第二天能不能干的问题，况且对于我这种拖延症患者，临时需要洗件衣服也是常事。烘干机完美解决了我们的穿衣自由，临时洗点脏衣服，从洗完到能穿的等待周期由一天起缩短至了一个半小时。我买的是石头 H1 Neo，感觉烘干效果也非常不错了，也没有遇到很明显的缩水问题。

其次就是回音壁，之前跟对象纠结很久要不要买，最后还是买了一套三星的 Q990D。最大的感受就是，这个东西在店里和家里效果是完全不一样的。虽然线下店里可以让你对整体音质有个大概的了解，但各种细节（尤其是环绕感）还需要在家里安静的环境才能体验到。打游戏或者看杜比全景声电影时犹如身临其境，这种体验与之前只用一套 HomePods 完全不一样。对于音频发烧玩家来说，这样的配置应该仍算一般的，但对于普通人来说感觉完全足够了。另外也可能是太久没看电视了，才发现现在的电视与配件的联动可以做到这么好，电视、音响和 Apple TV 的遥控器都可以互相控制，甚至可以控制 PS。

# 投资 101

工作这么长时间，其实一直没有认真研究过投资理财，感觉闲置资金的利用率太低了。与其让它们持续贬值，不如充分利用起来。所以从 2024 年底开始，我就在慢慢学习各种投资知识。最开始接触了一点加密货币，购买了少量的 USDT 和 BTC，不过后来不小心碰了邪修，亏了些钱。后面冷静下来以后，决定还是做一个长期投资者。

我现在的主要成长型投资标的就是美股、黄金和比特币。投资组合的分配上，美股占总资金 40%，黄金和比特币总共占 10%，而剩下的资金留作消费、应急和更灵活的稳健理财（比如债券类基金）。

在选股方面，我一直比较信奉人算不如天算。我认为跑赢大盘不是不可能，但对于我来说一定不可持续，因为我不希望投资变成一个需要消耗我大量精力的事情。所以现在包括以后，在投资标的上，美股指数基金会变成我主要的仓位。在日常操作上，保持好 DCA 节奏，不急不躁地长期持有下去，慢慢让它变成我的一个被动收入。不过目前我也有一半仓位在个股上，比如 NVDA、AAPL、TSLA 等等。大部分都是年初建仓的，也算是有一些个人信仰驱动吧。

![Portfolio](https://images.cyandev.app/2025-update-notes/ibkr_scr.png "Portfolio;1692x456;LDS~t|~WS~?v~qR*njn%tRR*n%e.")

回顾一下今年的投资表现：美股整体收益 14%，黄金收益 39%，比特币建仓较晚收益 -11%。我深知投资是一件长期甚至一辈子的事情，所以一年的表现也只是一个快照作为参考，说明不了什么。

另外感觉自己还是比较幸运的，在国内，好的投资渠道都在慢慢关闭，而我顺利地上了这趟末班车。所以不管做什么事情真的都要趁早，才能不给自己留下遗憾。

当然，免责声明还是要有的：每个人都有每个人的投资风格和风险偏好，上面所说的只是我不成熟的个人想法，不构成任何投资建议。

# 再出去看看世界

今年去了趟新加坡，这是我继日本之后第二次出国旅游。因为免签，所以旅行计划上也可以相对随意一些。不过刚入境时还是遇到了尴尬的事情，我被关小黑屋人工审验了一波，好在应该是乌龙了，后面顺利入境。

之前一直听说新加坡很小，应该逛不了几天。但实际去了才发现，如果仔细玩的话，一周都不一定够。我们这次只走马观花了一些著名景点，比如樟宜机场、鱼尾狮、Gardens By The Bay 和圣淘沙等等。虽然都在一个国家，但不得不说这些景点各具特色，感觉文化上非常多元。Talk is cheap，还需要大家亲身感受。

# 技术与工作

今年在工作上有一点拥抱变化了，因为公司战略多次调整，我们每个人的工作内容都有比较大的变化。首先是 leadership 层计划使用 Web 技术重构整个客户端，因为最终对我们的影响未知，mgr 希望大家做好转方向的准备。

我因为之前有 Web 开发的经验，所以年中被调去兼职一个新项目的开发。这个项目简单来说就是一个内部开发平台，使用了 Tauri + Vue.js 的技术栈。我前期负责了一些 Rust 层基建的部分，后面 mgr 让我 lead 了其中某个业务模块，我与小组成员花了一个多月的时间开发完了 MVP 版本，然后又被抽调走了。在这期间我自己也写了几个可以供其他项目复用的轻量库，打算时机成熟后全部开源，目前已经开源了一个异步任务调度库 [ninmu](https://github.com/unixzii/ninmu)。

接下来我又加入了 Web 重构计划，不过戏剧的是我的组织关系并没有调整。这意味着假设公司以后有 layoff 计划，我的 HC 也是算在客户端团队里的。这也意味着我给 Web 项目贡献的每一行代码，都有可能变成之后射向我自己的子弹。这个 Web 项目由成熟的前端团队主导，技术栈是 React，但他们自研的状态管理框架并不是很好用。奈何我在这个团队人微言轻，只能闷头写好自己该写的代码。

从今年开始，感觉我在公司的劲头慢慢消散了，越来越只把它当作一份赚钱的工作，更不用提什么热爱了。主要的原因我想应该是公司第二个大变化，现在全公司上下推行 all in AI 战略，但又像无头苍蝇一样完全找不到方向，老板甚至还用美国已经开始 996 文化为由取消了现在的 WFH 政策。这些变化使得我对公司的好感全无，工作态度也日渐消极（有点像上家公司离职之前的感觉）。用美国人的说法就是 **Quiet Quitting**，但我很享受这种工作与生活隔离的状态，我认为这是所有人都应该拥有的理想状态。热爱工作对于公司来说是一件 nice to have 的品质，而热爱生活对于个人来说更是一件理应获得的权利。所以有的时候很羡慕北欧的人们，他们不必为生计奔波，从而拥有为爱发电的底气。

工作之余，我也写了一些自己喜欢的小玩意儿。年初参与了砍砍的 FlowDown 项目，产出了一些动画和 UI 相关轮子。而今年对开源社区贡献比较多的是一些 Rust 项目，比如 Zed 和 GPUI。亲手解决一些自己遇到的问题，然后看它们出现在日用软件的 release notes 里，这种连结的感觉很奇妙。希望来年能更多地参与开源，让自己的价值辐射更广的地方。

# But what about AI?

等一下，怎么全文到现在还没提到 AI 呢？因为说实话，我今年真的没有怎么用 AI，我为 AI 而产生的付费约等于 0。甚至就在写完这篇文章后，我也没有用 AI 做任何优化。我的文笔一向不好，但我希望留下多一点真诚，少一些修饰。

我知道，这时很多人会认为我已经与时代脱节了，就像现在还在抱着 C 和汇编的老程序员一样，我自己有的时候刷 Twitter 都会有一些 fomo 的情绪。但现实中，我真的没有依赖 AI 太多。公司提供了 Claude 和 Cursor 的企业账号，我偶尔会在工作中使用。它们一定程度确实能够提升我的工作效率，对此我很感激。平时我也会问 ChatGPT 一些关于投资、历史以及各种技术的问题，它确实也帮到了我很多。但我使用的都是免费版，而这对我来说完全足够。

我不否认 AI 现在能力非常强大，它们写的代码有时候应该比我好，但我还是比较享受写代码的过程的。就像一个围棋手应该享受下棋的过程，而不是看 AI 赢得别人的过程吧。但的确有很多人是享受结果的，对于他们来说，AI 真的非常重要。Bret Victor 曾经说，[探索新事物需要一种媒体](https://worrydream.com/MediaForThinkingTheUnthinkable/)，AI 现在就是一种媒体，连接起了创意和成果。所以不管外部世界如何变化，始终去追求自己热爱的东西吧。

我相信 AI 在未来一定会成为解放生产力的工具，前景非常广阔。所以我也很希望自己以后有机会可以参与到 AI 的建设之中，在那里继续发挥自己的兴趣。
